Conheça um pouco!


Somos uma empresa focada na criação de softwares, como este que você está vendo agora. Esta aplicação foi desenvolvida utilizando linguagens como: Java, JavaScript, CSS, MySQL e HTML.


Com este sistema, conseguimos realizar operações CRUD (GET, POST, PUT, DELETE), graças ao framework Java denominado Spring Boot. Mas o que tornou tudo mais simples e organizado foi o tipo de arquitetura utilizada. Neste caso, foi adotada a arquitetura em camadas,com o back-end estruturado em pastas para cada responsabilidade. Por exemplo:


  • Controller: utilizada para definir as rotas da aplicação;

  • Service: responsável pela regra de negócios (lógica);

  • Model: como o próprio nome diz, é responsável pelos modelos das tabelas do banco de dados;

  • Além de outras camadas como DTOs e DAOs.
